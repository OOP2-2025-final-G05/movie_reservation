# アプリ名: 映画予約管理システム

## 概要:

### トップページ(統計ページ)：

**予約人数の表示:** 円グラフに映画ごとの予約人数を数字とともに割合で分けて表示

**映画別売上の表示:**　棒グラフにその月の映画の売上を表示

**月別総売上推移の表示:**　折れ線グラフに月別の総売上を表示

**各ページへの遷移:**　ボタンをクリックすることで、登録ページにアクセスできる

### 登録ページ：

**ユーザーの登録:** ユーザー名の登録・年齢の登録・区分分け

**映画の登録:** 映画の登録・映画の価格の登録

**映画の予約:** 予約したユーザー名・映画名・区分を参照した値段・予約した時間の表示・または予約の削除
