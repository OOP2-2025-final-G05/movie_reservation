# アプリ名: movie_reservation

概要:

トップページ(統計ページ)：

**動員数の表示:** 円グラフに映画ごとの動員数を数字とともに割合で分けて表示

登録ページ：

**ユーザーの登録:** ユーザー名の登録・年齢の登録(子ども・大人・シニアの自動振り分け)

> ユーザーID, 名前, 年齢

**映画の登録:** 映画の登録・映画の価格の登録(子ども価格・大人価格・シニア価格の表示)

> 映画ID, 名前, 価格

**映画の予約:** 予約したユーザー名・映画名・値段・予約した時間の表示

> ユーザーID, 映画ID
